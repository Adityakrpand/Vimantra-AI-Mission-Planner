# User Guide

This folder contains operator-oriented documentation.

Version 1.0 user workflow:

1. Start backend and frontend.
2. Add or load mission waypoints.
3. Save the mission.
4. Start PX4 SITL.
5. Connect to PX4.
6. Upload the saved mission.
7. Arm the simulated drone.
8. Start the mission.
9. Monitor telemetry.

See [../px4-sitl-workflow.md](../px4-sitl-workflow.md) for verification steps.
